# Wingspan Card Designer

A browser-based tool for designing custom [Wingspan](https://stonemaiergames.com/games/wingspan/) bird cards.

### Credits

The bird card HTML structure and CSS styling are derived from [WingSearch](https://navarog.github.io/wingsearch/) by navarog.

### Contribute

Feel free to fork the repo and make pull requests, or just leave suggestions in [issues](https://github.com/Touniouk/Wingspan-Card-Designer/issues)
